# Images Directory Structure

## Organization

### `packages/`
Photos of your proposal packages (decorations, setups)
- Use descriptive names: `minimalist-package.jpg`, `marry-me-lights.jpg`, etc.
- Recommended size: 800x600px or 1200x900px
- Format: JPG or PNG

### `gallery/`
Gallery photos showcasing past proposals
- Use descriptive names with numbers: `proposal-01.jpg`, `proposal-02.jpg`, etc.
- Recommended size: 1200x800px
- Format: JPG

### `locations/`
Photos of proposal locations in Paris
- Use location names: `eiffel-tower.jpg`, `trocadero.jpg`, `montmartre.jpg`
- Recommended size: 800x600px
- Format: JPG

### `hero/` (optional)
Background image for the hero section
- Use: `hero-background.jpg`
- Recommended size: 1920x1080px (Full HD)
- Format: JPG (optimized for web)

## Naming Convention
- Use lowercase
- Use hyphens for spaces: `my-image.jpg`
- Be descriptive: `large-heart-package.jpg` instead of `IMG001.jpg`
- Add version numbers if needed: `package-v2.jpg`

## Image Optimization
Before uploading images:
1. Resize to recommended dimensions
2. Compress for web (tools: TinyPNG, ImageOptim)
3. Target file size: <500KB per image
